Header tagline: Sincronizar productos, compras y clientes con Perfit Email Marketing.

Capta la información de tus compradores y productos desde Perfit para realizar envíos automáticos o masivos.

== Description ==

Perfit es la aplicación de email marketing te permite enviar a cada contacto lo que desea recibir.

A través del plugin de Perfit para WooCommerce, podrás sincronizar automáticamente tus productos y compradores hacia Perfit Email Marketing, para realizar campaña de email automation y de envíos masivos.

Para poder activar el plugin debes obtener el API key de tu cuenta de Perfit. Esta funcionalidad está disponible sólo para planes pagos.

Uso de servicios de terceros (third party services):
Este plugin utiliza los serivicios de Perfit (https://api.myperfit.com) para realizar la sincronización de los datos de productos y compradores.
Para conocer las políticas de privacidad y términos del servicio, visita https://docs.myperfit.com/es/collections/147168-terminos-y-privacidad.


== Installation ==

La manera más sencilla es utilizar el instalador de plugins de WordPress.

Una vez instalado en tu sitio, debes ir a los ajustes de WooCommerce (desde el menu de la izquierdo), seleccionar el nuevo tab PERFIT e ingresar el API key de tu cuenta de Perfit. Esta funcionalidad está disponible sólo para planes pagos.
